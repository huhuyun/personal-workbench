页面导航

页面顶部有四个导航按钮和一个搜索框：

按钮作用
首页显示全部工作和生活内容
工作只看工作相关待办、日程、备忘
生活只看生活相关待办、日程、备忘
日历跳转到月历视图

搜索框可全局搜索待办、日程、备忘的标题和内容。

待办事项

添加待办

在待办输入框输入标题，可设置以下属性：

范围：工作/生活 优先级：高/普通/低 截止日期 进度：0%~100% 所属项目/主题 标签（逗号分隔）

分组展示

点击右上角可切换两种视图：

  • 按截止日期 分为：已逾期 → 今天 → 本周 → 以后 → 无截止日期 → 已完成
  • 按项目主题 按所属项目分组展示
折叠功能：所有分组都支持点击标题展开/收起。"已逾期""今天""本周"默认展开；"以后""无截止日期""已完成"默认折叠，减少视觉干扰。

操作

操作方式
标记完成/取消完成点击待办左侧复选框
编辑点击"改"按钮
删除点击"×"按钮
进度与完成双向联动：标记为已完成时，进度自动变为 100%；反之，编辑时将进度设为 100%，待办自动标记为已完成。

日程安排

添加日程

在日程表单中填写：

标题（必填） 开始时间（必填） 结束时间 范围：工作/生活 所属项目/主题 标签

展示规则

  • 日程列表按日历选择的月份过滤，日历切到 8 月只看 8 月日程
  • 按开始时间升序排列
  • 自动标记状态：进行中 / 即将开始 / 已结束

日历视图

  • 按月展示日程和有截止日期的待办
  • 点击"上月""下月"切换月份
  • 切换月份后，下方日程列表同步过滤为对应月份
  • 每天格子内显示当日日程和待办的数量标记

项目/主题

项目/主题用于给待办、备忘、日程建立上下文关联。

默认折叠：项目/主题管理模块默认折叠，点击标题可展开查看和管理项目列表。

操作

操作说明
新增输入名称，点击"添加"
编辑点击"编辑"修改名称
删除仅可删除无关联事项的项目
排序点击"上移""下移"调整顺序
"未归类"是默认项目，不可删除或排序。

信息备忘

添加备忘

填写标题（必填）和内容（必填），可设置：

范围：工作/生活 所属项目/主题 标签

适合记录会议纪要、资料链接、购物灵感等。

数据管理

页面底部"本地数据管理"区域（可折叠）：

功能说明
填充示例数据一键生成演示用待办、日程、备忘
导出 JSON下载全部数据为 JSON 文件
导入 JSON从备份文件恢复数据
清空数据删除所有本地数据（不可撤销）

云端同步

注册与登录

  • 在"PWA 与云端同步"区域输入邮箱和密码
  • 点击"注册"创建账号，需到邮箱确认
  • 点击"登录"使用已有账号

自动同步

默认开启 勾选"自动同步"后：

  • 自动上传：每次新增/编辑/删除数据后，延迟 3 秒自动上传到云端
  • 自动拉取：页面加载或登录成功后，自动从云端拉取最新数据
  • 状态栏显示"自动同步已开启 · 上次同步 HH:MM"

手动同步

  • "上传本地到云端"：强制上传当前数据
  • "从云端拉取"：强制拉取云端数据（会覆盖本地）
多端使用：电脑改了数据 → 自动上传 → 手机打开页面 → 自动拉取，无需手动操作。

安装到手机/桌面

手机安装（PWA）

  • iPhone：用 Safari 打开页面 → 分享 → 添加到主屏幕
  • Android：用 Chrome 打开页面 → 菜单 → 安装应用

桌面安装

  • 用 Chrome/Edge 打开页面 → 地址栏右侧安装图标 → 安装
安装后像原生 App 一样使用，全屏启动，无需浏览器地址栏。
